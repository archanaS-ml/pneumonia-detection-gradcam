A deep learning system that classifies chest X-rays as Normal or Pneumonia, using transfer learning with ResNet18 and Grad-CAM for visual explainability. Deployed as an interactive Streamlit web app.

Pneumonia is a leading cause of death in children under 5 worldwide. Rapid, accurate detection from chest X-rays can support faster clinical decisions, especially in resource-limited settings. This project explores whether a CNN can reliably detect pneumonia while remaining interpretable to clinicians.

[Chest X-Ray Images (Pneumonia)](https://www.kaggle.com/datasets/paultimothymooney/chest-xray-pneumonia) — Kaggle, ~5,800 labeled chest X-rays (Normal / Pneumonia).

1. **Baseline CNN** (3 conv layers, trained from scratch) — established a reference point.
2. **Transfer Learning with ResNet18** (ImageNet pretrained, fine-tuned final layer) — improved generalization significantly.
3. **Grad-CAM** — visualized which image regions influenced each prediction, to audit model behavior.


| Model | Train Accuracy | Test Accuracy |
|---|---|---|
| Baseline CNN | 98.77% | 76.44% (overfit) |
| ResNet18 (Transfer Learning) | 94.79% | **89.42%** |

- Pneumonia Recall: **94%** (prioritized to minimize missed diagnoses)
- Normal Recall: 82%
- Overall Accuracy: 89.4%

Grad-CAM visualizations revealed the model does not always focus purely on lung opacity regions — in some cases, attention was drawn to central chest structures or peripheral areas. This is a known limitation of CNNs trained on limited medical datasets, where models can learn shortcut correlations rather than clinically meaningful features. This finding highlights why explainability tools are essential before any real-world deployment of medical AI systems.

- Grad-CAM (pytorch-grad-cam)
